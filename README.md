## Featured projects

### Real-time ETL pipeline
> End-to-end streaming pipeline ingesting 1M+ events/day.

- Kafka → Spark Structured Streaming → PostgreSQL
- Automated alerting & data quality checks
- Deployed on AWS ECS with Docker + Kubernetes

**Stack:** `Python` `Apache Kafka` `Apache Spark` `PostgreSQL` `Docker` `AWS`

---

### Sales analytics dashboard
> Interactive BI dashboard connecting 5 data sources with live KPI tracking.

- Built on Azure Synapse Analytics
- Grafana dashboards with real-time metrics
- Row-level security & multi-tenant support

**Stack:** `Azure Synapse` `Grafana` `SQL Server` `Python` `dbt`

---

### Churn prediction model
> ML pipeline for customer churn with 91% accuracy.

- Feature engineering pipeline with Pandas + scikit-learn
- Model versioning with MLflow
- Deployed via Docker + AWS SageMaker

**Stack:** `Python` `scikit-learn` `PyTorch` `MLflow` `Docker` `AWS SageMaker`

---

### Data warehouse migration
> Migrated on-prem SQL Server warehouse to Azure Synapse.

- 60% cost reduction
- 3× query performance improvement
- Full dbt transformation layer

**Stack:** `SQL Server` `Azure Synapse` `dbt` `Python` `Azure Data Factory`
